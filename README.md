# MysteryQuest  
2017年に発売した謎解きアクションゲームに使われた化学エンジンの機能をUnityで再現しそれを活かして開発したゲームプロジェクトになります。  
# 特徴  
化学エンジンの機能の大まかな現在の仕様  
□マテリアル：エレメントによる影響を受けるオブジェクト  
・木    
・石  
・鉄  
・氷  
□エレメント：マテリアルに影響を与えるオブジェクト  
・火  
・氷

#　ビルド手順  
・zipでダウンロード  
　zipでダウンロードする場合はダウンロードしたファイルを解凍しUnityHubでプロジェクトを追加し開くことができます。  
・Cloneでダウンロード  
　ForkやDesktop(それ以外でも可)のサーバーを使用してCloneするならMysteryQuestの「Code」でHTTPSをコピーし  
　サーバーで張り付けてCloneすればプロジェクトを確認することができます。
